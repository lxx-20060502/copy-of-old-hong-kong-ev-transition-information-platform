Client Interface

For Vehicle Market Statistics For EV-Charging Network

The Client Interface is the user-facing entry point of the system. It is designed to present dataspace services in a clearer and more accessible way, so users can understand what a data product is, what it can provide, and how they can request it without dealing directly with technical responses or backend fields.

In this page, the main function is to turn complex backend operations into a simple frontend experience. Users can browse available services, read concise descriptions of assets, start a request, and follow the progress of the process through visible status updates such as discovery, agreement initiation, transfer, and delivery. The page focuses on readable actions and understandable results instead of raw API outputs.

The interaction method is different from a technical demo page. Users do not manually trigger each protocol step or inspect detailed returned payloads. Instead, they interact through straightforward buttons and guided status feedback. Once a request is started, the system handles the workflow in the background and only exposes the information that is meaningful to the user, such as current stage, service details, policy summary, and final controlled result.
