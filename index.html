Hong Kong's EV Transition Information Platform

CSC4240 Project Demonstration

Project Overview

Vision

We envision a smarter and more connected EV ecosystem in Hong Kong, where public-sector data can be discovered and used across organizations without sacrificing data ownership, trust, or governance. As electric mobility continues to grow, data should not remain isolated within departmental silos. Instead, it should support collaboration between government and industry in a secure, interoperable, and sovereignty-preserving way.

Mission

Our mission is to demonstrate how a federated dataspace can enable practical and governed data sharing for real EV-related use cases. In this project, the Transport Department and the Environmental Protection Department act as independent data providers, while an EV manufacturer and a map service provider act as data consumers. Through a Federated Catalog and policy-based access control, we create a realistic environment in which multiple parties can discover and exchange data under clear contractual terms.

Goal

The goal of this project is to solve three key challenges in Hong Kong’s EV data ecosystem: fragmented data access, limited cross-organizational interoperability, and weak control over data usage after sharing. By building a federated data-sharing architecture, we enable unified asset discovery, governed data exchange, and practical applications such as market analysis and real-time charging navigation. The project also shows how real public data can be turned into usable digital assets for cross-sector collaboration.

What Problem We Solve

Our project enables public EV-related data to create practical value across organizational boundaries. By establishing a federated and governed data-sharing environment, we allow businesses to discover relevant assets more efficiently, access them through controlled exchange processes, and apply them to real operational scenarios. This supports use cases such as market intelligence for EV manufacturers and charging navigation services for mobility platforms, while giving data providers clear control over access conditions and usage terms. In this way, the project transforms public data from static information resources into actionable business assets for innovation and service improvement.
